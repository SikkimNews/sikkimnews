# 📰 Sikkim News

Official website for **Sikkim News** — a digital media brand dedicated to delivering reliable, timely, and community-focused news from across Sikkim.

---

## 🌐 About

This project is a static website built using:

- HTML5
- CSS3
- JavaScript

It serves as the official online presence of Sikkim News, providing visitors with information about the organization, advertising opportunities, contact details, and links to our social media platforms.

---

## ✨ Features

- Responsive Design
- Modern User Interface
- Hero Banner
- About Section
- Facebook Feed Integration
- Why Choose Us
- Advertise With Us
- Submit News
- Contact Section
- Social Media Links
- Privacy Policy
- Terms of Use
- Custom 404 Page
